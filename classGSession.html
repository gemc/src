Detailed Description

An instance of GSession, replacing G4UIsession, is passed to the G4UImanager pointer in gemc.cc main with: SetCoutDestination(new GSession)
The constructor will create the files for the log and error destinations: MasterGeant4.log and MasterGeant4.err
